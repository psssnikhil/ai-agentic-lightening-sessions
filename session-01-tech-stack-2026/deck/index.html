Agent in 30 seconds

A model in a loop — with tools and a goal

The one-line test
Who decides the next step — you, or the model?
If the model loops and calls tools → it's an agent.
Not a chatbot
A loop that can decide its own next step and call tools until a goal is met.

    for call in tool_calls(resp):
