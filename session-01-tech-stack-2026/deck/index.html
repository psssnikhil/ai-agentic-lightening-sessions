Definition

What makes it an agent?

The one-line test
Who decides the next step — you, or the model?
If the model loops and calls tools → it's an agent.
Not a chatbot
A loop that can decide its own next step and call tools until a goal is met.

Live demo

All seven components in one script

# hello_stack.py — every component, labelled
messages = [{"role":"user","content": TASK}]        # 2 MEMORY

for step in range(MAX_STEPS):                        # 5 GUARDRAILS
    resp = chat(messages, system=SYSTEM, tools=TOOLS)# 1 MODEL + 4 ORCHESTRATION
    log(resp.usage, latency)                         # 7 OBSERVABILITY

    if resp.stop_reason != "tool_use": break
    for call in tool_calls(resp):
        if call.name in RISKY: approve(call)          # 6 HUMAN-IN-THE-LOOP
        result = run_tool(call)                       # 3 TOOLS
    messages.append(results)                          # loop → MEMORY grows
